# 01. Drum Project.

This is the Drum Project, where you can type on any of the keys in your keyboard and it'll produce a sound.
Come on, try it and have fun.

This project will only allow you to type these keys:

* A.
* S.
* A.
* G.
* H.
* J.
* L.
